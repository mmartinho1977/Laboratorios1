Sumario:
This project consisted in creating 1 bash script for Linux Terminal, that automtically creates a set of directories for a local video library named "Videoteca".

Neste projeto criei um script bash para gerir a minha biblioteca de reviews de filmes
Para correr este script é preciso dar permissão de execução ao ficheiro.

Comandos:
 script.sh -i
 -Inicia a pasta videoteca com algumas categorias
 
 script.sh -a
 -Serve para adicionar um novo filme
 
 script.sh -l
 -Lista todas as categorias e filmes
 
 script.sh -c
 -Comprime a pasta Videoteca
 
 script-sh -d
 -Descomprime a pasta Videoteca se existir na mesma diretoria
