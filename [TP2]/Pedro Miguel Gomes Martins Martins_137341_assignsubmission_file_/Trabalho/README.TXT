Para a correta utilização dos scripts o utilizador deve iniciar com o script da estrutura, pois trata-se de um script dinâmico onde o utilizador escolhe se deseja uma videoteca predefinida ou uma videoteca costumizada, ou seja para a funcionalidade dos restantes precisa de a criar primeiro.

Script-estrutura: Toda a informação escolhida será exposta no ambiente de trabalho.

Script-unzip: Script onde o utilizador escolhe se deseja zip ou unzip, depois da criação da videoteca predefinida.

Script-abc:Script onde ordena alfabeticamente os tops casters dos filmes e series da videoteca predefinida.

Script-unzip e Script-abc apenas funcionam para a videoteca predefinida pois trata-se um nome fixo. Quando o utilizador cria a videoteca costumizada fica com o nome à escolha e os scripts não têm acesso ao nome da sua videoteca.
