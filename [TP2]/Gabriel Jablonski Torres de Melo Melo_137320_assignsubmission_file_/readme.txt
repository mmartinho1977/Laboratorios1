REQUISITOS PARA A SCRIPT FUNCIONAR:

- Os arquivos precisam estar na mesma pasta, mas não importa qual a pasta;
- É necessário digitar exatamente como a script lhe pede, caso contrário, o programa poderá finalizar e não será mais reutilizável.
- É necessário também ter instalado os pacotes da extenção .zip (para fazer isso digite no terminal "sudo apt-get install zip unzip")


CASO A SCRIPT FALHE POR DIGITAR ALGO ERRADO

- Será necessário cancelar a execução da script(Crtl+C) e deletar a pasta VideotecaTemp
- Após isso a script pode ser executada novamente sem problemas.
