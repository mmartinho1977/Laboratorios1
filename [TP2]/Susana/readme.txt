Indicações para que o script seja corretamente executado.

Todas imagens e a música terá de estar na sua pasta pessoal e soltas, de modo a que a localização das mesmas seja /home/o seu utilizador 
Antes de executar o sript verifique se tem estas apps instaladas : mpg123 (para reprodução de mp3) xdg-open (para abrir imagens no terminal) caso não tenha fazer o seguinte:
Entrar em modo root digitando sudo -s
digitar sudo apt-get install mpg123 
sudo apt-get install xdg-utils em modo root
digitar exit para sair do modo root

Para depois executar o srcipt digitar chmod +x videoteca.sh e enter

digitar ./videoteca.sh
