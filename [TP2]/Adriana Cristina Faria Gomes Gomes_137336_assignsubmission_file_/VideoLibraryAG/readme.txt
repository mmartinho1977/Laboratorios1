Neste ficheiro serão descritas as instruções para o correto funcionamento das scripts desenvolvidas no Terminal-Ubunto.

## Informação ## 

A pasta VideoLibraryAG deve ser descompactada.

Na pasta VideoLibaryAG devem constar, além deste documento, dois ficheiros de extensão sh (criar_VideoLibrary.sh e manipulacao.sh) e uma pasta com o nome VideoLibrary1. 

A pasta VideoLibrary1 é a pasta inicial, ou seja, o seu conteúdo é igual à da pasta criada a partir da execução da script "criar_VideoLibrary.sh" e sem qualquer alteração.


## Execução das scripts ##

As scripts devem ser executadas respeitando a seguinte ordem de passos:

1. Executar a script "criar_VideoLibrary.sh".

2. No ficheiro manipulacao.sh definir as variáveis (considerar diretorias após descompactar a pasta VideoLibrary): 
	 dirtv - Diretoria para a pasta Type
	 dirtm - Diretoria para a pasta Movies
	 
3. Executar a scrpit "manipulacao.sh".
